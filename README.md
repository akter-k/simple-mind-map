# Simple Mind Map

A lightweight, interactive mind mapping application built with pure HTML, CSS, and JavaScript. No dependencies, no build tools - just open the HTML file in your browser and start creating your mind maps.


## Features

* Intuitive Interface: Create hierarchical mind maps with unlimited nesting levels
* Visual Organization: Color-coded branches with 5 distinct levels and smooth visual hierarchy
* Rich Content: Add codes and descriptions to each node for detailed documentation
* Collapsible Nodes: Expand/collapse branches to focus on specific areas
* Smooth Navigation: Pan and zoom with mouse controls (Ctrl + Wheel for zoom)
* File Management:
  * Save/Open files directly using File System Access API (supported browsers)
  * Export/Import as JSON for backup and sharing
* Bilingual Support: Switch between English and Russian interfaces
* Keyboard Shortcuts: Press Tab to quickly add child nodes
* Responsive Design: Clean, modern UI with hover effects and smooth transitions

## How to Use

1. Getting Started: Download MindMaps.html and open it in a modern web browser
2. Creating Nodes:
  * Click the "+" button to add child nodes
  * Press Tab while editing a node to create a child
  * Edit node titles and codes by clicking on them
3. Adding Details: Click the "i" icon on any node to add a detailed description
4. Navigation:
  * Drag the canvas to pan
  * Hold Ctrl and use mouse wheel to zoom in/out
  * Click "−" button to collapse node branches
5. Saving Your Work:
  * Use "Save" button (modern browsers) for direct file access
  * Use "Export (JSON)" to download a backup file
  * Use "Import (JSON)" to load previously saved maps

## Browser Compatibility
* Works in all modern browsers (Chrome, Firefox, Safari, Edge)
* File System Access API (Open/Save buttons) supported in Chrome/Edge
* All other features work universally across browsers

## Technical Details
* Pure vanilla JavaScript - no frameworks or libraries
* Single file application - everything in one HTML file
* localStorage-free - your data stays in files you control
* Responsive and accessible - clean semantic HTML structure

# Простой Mind Map
Легкое интерактивное приложение для создания интеллект-карт на чистом HTML, CSS и JavaScript. Никаких зависимостей, никаких инструментов сборки - просто откройте HTML-файл в браузере и начните создавать свои карты.

## Возможности
* Интуитивный интерфейс: Создавайте иерархические интеллект-карты с неограниченной вложенностью уровней
* Визуальная организация: Цветовое кодирование ветвей с 5 различными уровнями и плавной визуальной иерархией
* Богатое содержимое: Добавляйте коды и описания к каждому узлу для детальной документации
* Сворачиваемые узлы: Разворачивайте/сворачивайте ветви для фокусировки на конкретных областях
* Плавная навигация: Перемещение и масштабирование с помощью мыши (Ctrl + колесо для зума)
* Управление файлами:
  * Сохранение/открытие файлов напрямую через File System Access API (поддерживаемые браузеры)
  * Экспорт/импорт в формате JSON для резервного копирования и обмена
* Двуязычная поддержка: Переключение между английским и русским интерфейсами
* Горячие клавиши: Нажмите Tab для быстрого добавления дочерних узлов
* Адаптивный дизайн: Чистый современный интерфейс с эффектами наведения и плавными переходами

## Как использовать
1. Начало работы: Скачайте MindMaps.html и откройте в современном веб-браузере
2. Создание узлов:
  * Нажмите кнопку "+" для добавления дочерних узлов
  * Нажмите Tab при редактировании узла для создания дочернего
  * Редактируйте названия и коды узлов, кликая по ним
3. Добавление деталей: Нажмите иконку "i" на любом узле для добавления подробного описания
4. Навигация:
  * Перетаскивайте холст для перемещения
  * Удерживайте Ctrl и используйте колесо мыши для масштабирования
  * Нажмите кнопку "−" для сворачивания ветвей узла
5. Сохранение работы:
  * Используйте кнопку "Сохранить" (современные браузеры) для прямого доступа к файлам
  * Используйте "Экспорт (JSON)" для загрузки файла резервной копии
  * Используйте "Импорт (JSON)" для загрузки ранее сохраненных карт

## Совместимость с браузерами
* Работает во всех современных браузерах (Chrome, Firefox, Safari, Edge)
* File System Access API (кнопки Открыть/Сохранить) поддерживается в Chrome/Edge
* Все остальные функции работают универсально во всех браузерах

## Технические детали
* Чистый vanilla JavaScript - без фреймворков и библиотек
* Однофайловое приложение - всё в одном HTML-файле
* Без localStorage - ваши данные остаются в файлах под вашим контролем
* Адаптивное и доступное - чистая семантическая HTML-структура
